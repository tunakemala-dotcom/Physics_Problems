# Task 07 – Standing Wave Modes

## Problem Statement

A standing wave with four antinodes is produced on a string of length $L = 80 \text{ cm}$. What is the wavelength of this wave?

## Theory

A string fixed at both ends supports standing wave modes characterized by integer numbers of antinodes (or harmonic number $n$). 
The distance between two adjacent nodes is half a wavelength ($\lambda/2$). An antinode exists exactly midway between each pair of adjacent nodes. Therefore, $n$ antinodes correspond to $n$ half-wavelengths fitting perfectly into the length $L$.

$$
L = n \frac{\lambda}{2}
$$

## Step-by-Step Solution

Given values:
* Number of antinodes $n = 4$
* String length $L = 80 \text{ cm} = 0.80 \text{ m}$

Rearrange the mode equation to solve for wavelength $\lambda$:

$$
\lambda = \frac{2L}{n}
$$

Substitute the values:

$$
\lambda = \frac{2(0.80)}{4}
$$

$$
\lambda = \frac{1.60}{4} = 0.40 \text{ m}
$$

## Final Result

The wavelength of this standing wave is $0.40 \text{ m}$ (or $40 \text{ cm}$).

## Interpretation

Because four complete antinodes fit onto the string, exactly two full wave cycles ($2\lambda$) occupy the $80 \text{ cm}$ length. This indicates the string is vibrating in its fourth harmonic mode.
